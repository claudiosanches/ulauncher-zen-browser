# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-04-21

### Added
- Initial release of the Zen Browser Search extension.
- Support for searching bookmarks and history in Zen Browser.
- Support for both local (`~/.zen`) and Flatpak installations.
- Customizable search types: Bookmarks, History, or both.
- Configurable sorting methods, including Relevance (Frecency).
- Direct URL opening with automatic `https://` prefixing.
